# ZenFinance iOS App

**This is where the product lives.** ZenFinance ships exclusively as an iOS app (Expo / React Native) — see `PLAN.md` Phase 4.

This directory is a placeholder until the product phase begins. The app will be scaffolded here with:

- Expo (managed workflow) + TypeScript + Zustand
- Plaid Link iOS SDK for account linking
- RevenueCat SDK for App Store subscriptions
- Sentry for crash reporting
- EAS for builds and TestFlight distribution

The API it talks to is `apps/api` in this repo; shared Zod schemas/types come from `packages/shared`.
